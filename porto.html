Projects

Aplikasi Pemesanan Roti Bakar
Program ini dirancang untuk mengelola pemesanan menu makanan pada aplikasi sederhana berbasis teks, di mana pengguna dapat menginputkan data diri seperti nama dan nomor telepon, memilih rasa roti bakar, dan melihat total harga yang harus dibayar.
Kasir Warteq
Program ini adalah aplikasi kasir sederhana untuk Warung Makan (WARTEQ PADANG). Program menampilkan daftar menu makanan, memungkinkan pengguna memilih dan memesan beberapa item, menghitung total harga, menerima pembayaran, menghitung kembalian, dan mencetak struk pembelian ke file struk.txt.
